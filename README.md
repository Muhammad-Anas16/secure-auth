# 🔐 Authentication System

A secure, production-ready authentication system built with modern web technologies, implementing real-world authentication flows used in modern applications.

---

## ✨ Features

- 🔑 Email Sign Up & Login  
- 🔄 Forgot Password & Reset Password Flow  
- 🔐 Google OAuth Authentication  
- 🐙 GitHub OAuth Authentication  
- 🔒 Secure handling of user credentials  
- 🎯 Clean and user-friendly UI

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Authentication:** Email/Password, Google OAuth, GitHub OAuth  
- **Email Service:** Nodemailer  
- **Security:** Password hashing, token-based verification  
- **Version Control:** Git & GitHub  

---

## 🔐 Authentication Flows Overview

### 1️⃣ Email Authentication
- Users can sign up using email and password
- Secure login with credential validation

### 2️⃣ Forgot & Reset Password
- Password reset request via email
- Secure token-based password reset flow

### 3️⃣ Social Authentication (OAuth)
- Login with Google using OAuth 2.0
- Login with GitHub using OAuth 2.0
